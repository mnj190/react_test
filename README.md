# 생활코딩 React

## Bulid / Start

1) npm run start --> 다양한 기능을 포함하고 있지만 용량이 크다. 개발 시 사용
2) npm run build --> 사용된 기능만 포함하여 용량이 작다. 배포 시 사용

## Component

* 컴포넌트 생산 시 하나의 최상위 태그만 사용
